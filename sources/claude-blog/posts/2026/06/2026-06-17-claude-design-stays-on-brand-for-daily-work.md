# Claude Design 现在能在日常工作中保持品牌一致性

> 来源：Claude Blog，2026-06-17
> 原文链接：https://claude.com/blog/claude-design-stays-on-brand-for-daily-work
> 分类：Product announcements

## 核心要点

- Claude Design 现在能在多个项目中遵循你的设计系统（design system），与 Claude Code 流畅协作，支持在画布上直接编辑，并接入更多你已在使用的工具。
- 重建了设计系统导入功能，可从 GitHub 仓库、设计文件或原始上传引入一个或多个设计系统，Claude 会依据你的设计系统校验并修正输出。
- 新增管理员角色，可审批并锁定一套标准设计系统，确保产出符合公司规范。
- 通过 `/design-sync` 与 `/design` 命令，可在 Claude Design 与 Claude Code 之间无缝切换并保持同步，把设计移交为软件时从已有成果继续而非从截图重做。
- 全新编辑器提供细粒度控制与丰富的布局功能，并完成数百项稳定性修复。
- Claude Design 现与聊天、Claude Cowork、Claude Code 共享使用额度，平均每轮消耗更少 token，错误大幅下降。
- 可可靠导出为 PDF 和 PowerPoint，并接入 Adobe、Base44、Canva、Gamma、Lovable、Miro、Replit、Vercel、Wix 等连接器。

## 中文译文

Claude Design 现在能在多个项目中遵循你的设计系统（design system），与 Claude Code 流畅协作，支持你直接在画布（canvas）上编辑，并接入更多你已在使用的工具。它在 Claude 桌面应用的侧边栏中拥有了新的入口；你也可以在 claude.ai/design 访问它。

我们的目标与之前一致：让你探索比截止期限通常允许的更多方向，并让交付给打磨或开发环节的过程尽可能无缝。这个目标如今服务了更多的人。Claude Design 在第一周就有超过一百万人使用，而你的反馈正在塑造我们接下来要构建的内容。

## 用你的设计系统组件来构建

我们重建了设计系统导入功能，以提供更高的灵活性并提升精确度。你可以从 GitHub 仓库、设计文件或原始上传中引入一个或多个设计系统。Claude 会用你的组件来构建，对照你的设计系统检查其输出，并在你看到之前进行修正。对于更大的团队，新的管理员（admin）角色可以审批一套标准系统并锁定编辑权限，从而让产出始终符合公司准则。

## 在设计与代码之间切换

现在，在 Claude Design 与 Claude Code 之间切换工作的同时保持成果同步变得更加容易，相关更新于今日陆续推出。使用 `/design-sync` 拉入你的设计系统，这样你在 Claude Design 中构建的一切都从你已有的组件出发。当一项设计准备好成为软件时，你可以将其移交给 Claude Code，它会从你已有的成果继续，而不是从一张截图重新开始。

更喜欢从 Claude Code 开始？你可以使用 `/design` 在不离开终端的情况下创建、编辑和同步设计项目。将设计导入你的代码库、把你的代码变成一个实时原型，或让 Claude 把一个项目从头贯通到底。

> “Claude Design 让我同时拥有前沿模型的智能，以及久经验证的设计工具的功能与能力。每当我为 Tenex 的网站、新的品牌资产或演示文稿构思设计方向时，Claude Design 都是我首先打开的地方。平易近人的用户体验与出色的品味和设计直觉相结合，是它成为我技术栈核心部分的原因。而 Claude Design 与 Claude Code 之间的移交，让从原型到生产的过程无缝衔接。” —— Alex Lieberman，Morning Brew 与 Tenex 联合创始人

## 在日常工作中更稳定

我们的新编辑器让你对设计中的每个元素都拥有直接、细粒度的控制。全新、丰富的布局控制让你能够拖动、调整大小和对齐元素。数百项稳定性修复让编辑器在真实使用中更加可靠。

Claude Design 现在还与聊天（chat）、Claude Cowork 和 Claude Code 共享使用额度，因此大多数人将获得更大的创作空间，而那些触及上限的人遇到限制的频率会更低。如今平均每一轮使用更少的 token 即可达到相同的效果，错误也大幅下降。

## 从画布到你使用的工具

可靠地导出为 PDF 和 PowerPoint，或将你的成果发送到你已在使用的应用——连接器（connector）列表现在包括 Adobe、Base44、Canva、Gamma、Lovable、Miro、Replit、Vercel 和 Wix，更多目标平台即将推出。
