# Claude Cowork 产品指南

> 来源：Claude Blog，2026-06-05
> 原文链接：https://claude.com/blog/the-claude-cowork-product-guide
> 分类：Enterprise AI

## 导语

我们分享如何上手 Claude Cowork，包括能将你的想法转化为精致交付物的工作流、习惯和插件（plugins）。

## 核心要点

- Claude Cowork 是一款新的知识工作智能体（knowledge work agent），让你把工作委派给 Claude，从而专注于更具战略性和创造性的问题。
- 它运行于 Claude 桌面应用中，可读写本地文件，跨 Slack、Google Drive 等已连接应用工作，并将多步骤任务执行到真正的交付物，附带回溯到实际文件和消息的引用。
- 你只需描述目标、期望成果和节奏，Claude 便会规划步骤并完成工作，确保你全程参与。
- 产品矩阵帮助你判断何时使用哪种工具：聊天用于对话式起草，Claude Code 用于编程，Claude Cowork 用于跨应用的知识工作。
- 指南涵盖核心能力（本地文件访问、子智能体、长时运行工作、定时任务）、安装与系统要求、权限模型以及上手建议。
- 指南列出七种常见的 Claude Cowork 工作流，包括研究简报、会议准备和周期性报告。
- 你还可以使用插件来完成营销、产品管理等特定知识工作。

## 中文译文

大多数 AI 工具是对话式的。你提出一个问题，得到一个答案，而把这个答案转化为有用之物——一份演示文稿、一篇文档、一张电子表格、一封邮件——的工作仍需手动完成。

Claude Cowork 是我们新推出的知识工作智能体（knowledge work agent），它让你把这类工作委派给 Claude，从而专注于解决那些占据你时间、更具战略性和创造性的问题。Claude Cowork 在 Claude 桌面应用（desktop app）中运行，能读写本地文件，跨 Slack 和 Google Drive 等已连接应用工作，并将多步骤任务执行到真正的交付物，附带回溯到实际文件和消息的引用（citations）。你描述目标、期望成果和节奏，Claude 便会规划出步骤并完成工作，确保你参与到每一个环节。

为帮助你上手，我们整理了一份关于 Claude Cowork 的实用产品指南。我们分享了：

- Claude Cowork 与对话式 AI 工具的不同之处及其核心能力，包括本地文件访问、子智能体（subagents）、长时运行工作（long-running work）以及定时任务（scheduled tasks）

- 一个产品矩阵，用于理解何时使用何种工具：聊天用于对话式起草，Claude Code 用于编程，Claude Cowork 用于跨应用的知识工作

- 安装与系统要求、权限模型（permissions model），以及值得先尝试的内容

- 七种常见的 Claude Cowork 工作流，包括研究简报、会议准备和周期性报告

- 如何使用插件（plugins）来完成营销、产品管理等特定知识工作
